All chart functions are done with D3.js.

Go here for more information: https://github.com/d3/d3/blob/master/API.md

Here are some useful links for learning d3.v4:
  1) D3 Introduction - https://d3js.org/#introduction
  2) Thinking with Joins - https://bost.ocks.org/mike/join/
  3) Nested Selections - https://bost.ocks.org/mike/nest/
  4) General Update Patterns - https://bl.ocks.org/mbostock/3808218
  5) D3 In Depth - http://d3indepth.com/
  6) Simple Scatter Plot - https://bl.ocks.org/d3noob/6f082f0e3b820b6bf68b78f2f7786084
  7) Pan and Zoom - https://bl.ocks.org/mbostock/d1f7b58631e71fbf9c568345ee04a60e

A great place to find examples is bl.ocks.org
If you want to experiment with an example on bl.ocks.org go to blockbuilder.org.

  Take Mbostock's brush zoom example: https://bl.ocks.org/mbostock/34f08d5e11952a80609169b7917d4172
  And replace 'bl.ocks' with 'blockbuilder': http://blockbuilder.org/mbostock/34f08d5e11952a80609169b7917d4172
